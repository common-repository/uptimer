The UpTimer plugin for WordPress lets you see and analyse your website uptime and pagespeed on your dashboard and in details in the plugin's pages 

== Description ==

This plugin lets you see and analyse your website uptime and pagespeed on your dashboard and in details in the plugin's pages. That gives you full control over the availability of your webservices. 

* An account in [UpTimer.info](https://www.uptimer.info/ "UpTime Monitoring Service") is required. The account is free for one monitor, so you can start monitoring your uptime right now!
* Installation and configuration in 1 Step
* Dashboard Widget for quick stats about your websites


== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/uptimer.info` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the UpTimer.info->Settings to add your API key ( You can obtain your API key from your [Dashboard](https://www.uptimer.info/admin/user/profile/) -> API Credentials at UpTimer.info website )
